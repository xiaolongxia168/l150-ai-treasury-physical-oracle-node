## 🤖 自动化运营策略

### 策略1: 推广自动优化 🚀

**目标**: 最大化推广ROI，降低获客成本

**自动化规则**:
```
IF 推广ROI < 2 THEN
  降低出价 10%
  发送告警通知
  
IF 推广消耗速度 > 预算80% AND 时间 < 18:00 THEN
  降低出价 5%
  延长投放时段
  
IF 点击率 < 2% THEN
  标记需要优化创意
  生成创意优化建议
  
IF 转化率 < 5% THEN
  分析转化漏斗
  生成商品优化建议
```

**执行频率**: 每30分钟检查一次

---

### 策略2: 评价自动管理 ⭐

**目标**: 维护高评分，快速处理差评

**自动化规则**:
```
IF 新增差评 THEN
  立即发送告警 (短信+微信)
  自动生成回复草稿
  标记为紧急处理
  
IF 差评率 > 5% THEN
  生成差评分析报告
  识别共性问题
  生成改进建议
  
IF 待回复评价 > 5 THEN
  生成批量回复建议
  标记回复优先级
  
IF 评分下降趋势 THEN
  发送趋势预警
  生成提升方案
```

**执行频率**: 每5分钟检查一次

---

### 策略3: 库存和商品自动优化 🛍️

**目标**: 优化商品结构，提升转化率

**自动化规则**:
```
IF 商品浏览量 > 100 AND 转化率 < 1% THEN
  标记为"需要优化"
  生成优化建议 (价格/图片/描述)
  
IF 商品库存 < 10 THEN
  发送库存预警
  建议补货或下架
  
IF 商品退款率 > 15% THEN
  标记为"问题商品"
  分析退款原因
  生成改进方案
  
IF 有爆款商品 (销量TOP3) THEN
  生成爆款分析报告
  建议复制成功要素
```

**执行频率**: 每小时检查一次

---

### 策略4: 客户自动运营 👥

**目标**: 提升复购率，激活沉默客户

**自动化规则**:
```
IF 新客户首次消费 THEN
  标记为"新客户"
  3天后发送复购优惠券
  
IF 客户30天未消费 THEN
  标记为"沉默客户"
  发送召回优惠
  
IF 客户消费频次 > 3次 THEN
  标记为"忠实客户"
  加入VIP客户群
  提供专属优惠
  
IF 高价值客户 (消费金额TOP10%) THEN
  生成客户档案
  定期发送专属福利
```

**执行频率**: 每日执行

---

### 策略5: 竞品自动监控 🔍

**目标**: 了解市场竞争，及时调整策略

**自动化规则**:
```
IF 商圈排名下降 THEN
  分析竞品变化
  生成应对建议
  
IF 竞品价格变化 THEN
  发送价格变动提醒
  生成定价建议
  
IF 竞品有新活动 THEN
  抓取活动信息
  生成应对策略
```

**执行频率**: 每日执行
