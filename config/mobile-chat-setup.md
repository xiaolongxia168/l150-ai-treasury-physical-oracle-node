# 移动聊天配置
## 目标
让用户能从手机主动发消息给我，实现双向通信

## 方案对比

### 方案1: iMessage (推荐iPhone用户)
- 优点: 原生，无需额外app
- 缺点: 仅限苹果生态
- 状态: 等待完全磁盘访问权限

### 方案2: Telegram Bot
- 优点: 跨平台，全球可用
- 缺点: 需要科学上网
- 状态: 可随时部署

### 方案3: 飞书机器人
- 优点: 已配置，国内可用
- 缺点: 工作场景，不够私密
- 状态: 已运行

## 执行计划
1. 用户设置iMessage权限 → 测试收发
2. 如iMessage不可用 → 部署Telegram Bot
3. n8n作为可选增强（webhook中转）

## 浏览器自动化配置
- 工具: OpenClaw browser (Chrome扩展模式)
- 目标网站: Gemini, Manus
- 用途: 代表用户与这些AI对话，获取结果
