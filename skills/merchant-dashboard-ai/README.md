# 商家后台智能爬虫

自动抓取抖音来客和美团开店宝商家后台的所有数据。

## 快速开始

```bash
# 1. 确保 openclaw 浏览器正在运行
# 2. 在浏览器中登录抖音来客和美团开店宝
# 3. 运行爬虫

python3 scripts/smart_crawler.py
```

## 功能

- 自动发现所有菜单项
- 自动点击并抓取数据
- 保存为 JSON + 截图

## 数据位置

- JSON: `data/`
- 截图: `logs/`

## 已抓取示例

抖音来客: 21 页面 (店铺管理、商品与货架、订单管理等)
美团开店宝: 9 页面
