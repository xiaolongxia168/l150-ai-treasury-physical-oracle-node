```
┌─────────────────────────────────────────────────────────────┐
│                    智能诊断指标体系                          │
├─────────────────────────────────────────────────────────────┤
│  流量健康度    │  转化率健康度   │  收入健康度   │  口碑健康度  │
├─────────────────────────────────────────────────────────────┤
│ • 日访问量     │ • 访问-下单率   │ • 日GMV      │ • 好评率     │
│ • 流量趋势     │ • 下单-核销率   │ • 退款率     │ • 差评率     │
│ • 流量成本     │ • 复购率        │ • 客单价     │ • 评价回复率 │
└─────────────────────────────────────────────────────────────┘
```

### 问题自动识别规则

| 问题类型 | 识别条件 | 优先级 |
|---------|---------|--------|
| 🚨 流量暴跌 | 日访问量环比下降>30% | P0 |
| 🚨 评分危机 | 美团评分<60或差评激增 | P0 |
| ⚠️ 转化低效 | 访问-下单率<2% | P1 |
| ⚠️ 退款异常 | 日退款率>20% | P1 |
| ⚠️ 库存预警 | 热销商品库存<10 | P1 |
| 💡 增长机会 | 某商品转化率>均值2倍 | P2 |
| 💡 优化建议 | 竞品对比发现的差距 | P2 |

## 📈 三、运营建议生成系统

### 建议模板库

#### 流量提升建议
```
问题: 美团访问人数低于商圈均值
分析: 当前日均访问60人，商圈均值120人
建议:
1. 优化店铺关键词（密室逃脱、恐怖密室、团建等）
2. 投放推广通，预算建议¥50-100/天
3. 更新店铺头图和相册，提升列表页点击率
预期效果: 访问量提升50-80%
```

#### 转化优化建议
```
问题: 抖音访问-成交转化率偏低
分析: 22访问仅1单成交，转化率4.5%
建议:
1. 优化商品详情页，增加场景图片和视频
2. 设置限时优惠，制造紧迫感
3. 增加用户评价展示
预期效果: 转化率提升至8-10%
```

#### 口碑维护建议
```
问题: 美团经营评分57.5分（偏低）
分析: 低于商圈平均分65分
建议:
1. 主动邀请满意顾客评价
2. 差评24小时内必须回复
3. 针对差评问题进行整改
预期效果: 评分提升至65分以上
```
