## 🛠️ 使用指南

### 常用命令

```bash

cd ~/.openclaw/workspace/skills/merchant-dashboard-scraper

./merchant-assistant.sh deep

./merchant-assistant.sh analyze

# 生成完整报告
./merchant-assistant.sh report

./merchant-assistant.sh status

./merchant-assistant.sh logs
```
